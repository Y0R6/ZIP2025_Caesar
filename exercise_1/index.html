What HTTP status code do you see for index.html? What does it mean?

=>304 Not Modified is mean the requested resource has not changed since the last time it was requested.

=>101 Switching Protocols is mean the server agrees to switch communication protocols as requested by the client.

=>But if the page is from a real domain is gonna look like "https://www.example.com/".
